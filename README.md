# vtt2lrc

-----------------------
数字后缀代表版本迭代顺序

-----------------------
无后缀项目

*使用说明*

*1.将你想转化的vtt文件放入tar文件夹*

*2.点击run.bat*

*3.转化后的lrc文件会生成在res文件中*

注意：转化后tar中的全部文件会自动**删除**

-----------------------
含有terminal后缀的项目

在主函数中输入存放目标文件的文件夹

运行结果：将目标文件夹下所有vtt文件转换为lrc文件

-----------------------


# vl2txt

将目标文件夹中所有vtt或lrc文件转换为无时间戳的txt文件
